##포맷 스트링 취약점.
- 기본적으로 %x,%p / %n / %s(Uncontrolled format string) 이런 포맷스트링은 위험성을 지닌다.
- %x => 스택값 찍기/16진수의 스택값을 가져온다.
- %p => 포인터 주소를 인자로 가져온다.
- %s => 포인터가 가리키는 문자열 출력.
- %n => printf가 출력한 바이트 수를 포맷스트링 인자가 가리키는 주소에 write. (가장 강한 write)
- %hn, %hhn으로 정밀한 주소 조작으로 GOT overwrite도 한다.
